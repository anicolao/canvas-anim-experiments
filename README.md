# canvas-anim-experiments
A repository to experiment with canvas paint speed on the Raspberry Pi 5

## Bouncing Ball Experiment

A simple canvas animation experiment featuring a red bouncing ball with realistic physics.

### Features
- Full-page canvas with black background
- Bouncing ball with gravity and damping physics
- Real-time FPS counter
- Frame paint time measurement
- Performance stats displayed in lower right corner

### View the Demo
Visit the GitHub Pages site: https://anicolao.github.io/canvas-anim-experiments/

### Enable Chrome's FPS Meter
To see Chrome's native FPS meter overlay:
1. Open DevTools (F12 or Cmd+Option+I)
2. Press Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows/Linux)
3. Type "Show frames per second (FPS) meter"
4. Press Enter

### Testing on Raspberry Pi 5
Open the page in Chromium on your Raspberry Pi 5 to benchmark canvas animation performance.
